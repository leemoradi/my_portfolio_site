Voting Analysis: Georgia & Texas

This project explores partisan shifts and voter turnout in Georgia and Texas over recent election cycles. Using publicly available voting data, I cleaned and organized the information in Excel, wrote a Python script to calculate vote percentages, and visualized the results using a comparative bar chart.
