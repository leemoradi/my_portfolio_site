Voting Analysis: Georgia and Texas

This project explores partisan shifts and voter turnout in Georgia and Texas over recent election cycles. Using publicly available voting data, I cleaned and organized the information in Excel, wrote a Python script to calculate vote percentages, and visualized the results using a comparative bar chart. Georgia and Texas were chosen due to their contrasting electoral trends: Georgia‚Äôs recent swing toward Democratic candidates and Texas‚Äôs evolving but traditionally Republican base. By analyzing this data over time, the project sheds light on broader regional political dynamics and highlights the importance of voter engagement and demographic shifts.

Steps I Took:

  • Collected county-level election data for Georgia and Texas
  • Used Python and pandas to clean and analyze the datasets
  • Calculated vote percentages for each major party by year
  • Created visualizations with matplotlib
  • Exported results to Excel and included data visualization in the final output
  • Deployed a live demo of the project using GitHub Pages
